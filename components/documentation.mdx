# Atomic Design Model

Atomic design is

- atoms,
- molecules,
- organisms,
- templates,
- and pages concurrently working together to create effective interface design systems.

More on Atomic Design Model: https://atomicdesign.bradfrost.com/chapter-2/

# Server Components

- Next.js 13 Components are Server Components by default. When interactivity is needed, mark your file as Client Component with "use client" at the very top of your Component file.

More on Server Components: https://nextjs.org/docs/advanced-features/react-18/server-components
